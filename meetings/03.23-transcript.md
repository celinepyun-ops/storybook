## Summary

### Gallop CRM — Key Decisions

1. **Search tab restructure**: Product-based search becomes the main tab (replacing brand search). Brand search moves to a separate secondary tab.

2. **List feature**: Introduce a "List" concept in the left sidebar menu. Users can filter/sort products, select them, and save to named lists — similar to Apollo CRM's UX.

3. **Simplify search results**: Remove ranking scores and unnecessary data. Show only essential info: product name, price, and Amazon link.

4. **Brand enrichment on list add**: When a user adds items to a list, brand + people enrichment happens automatically (no per-item opt-in). Accept the token cost for now and collect early user feedback.

5. **Natural language search**: Add a natural language search bar (like a prompt) that auto-maps to filters. Users can then adjust filters manually if needed, or just use sorting. Inspired by Apollo's UX.

6. **Pricing model (MVP)**:
   - Brand enrichment cost covered by the company
   - Pricing based on **lead count** (not tokens): Basic = 100 leads, Pro = 500 leads
   - MVP uses **plan upgrades only** (no à la carte token purchases yet)
   - Apollo API usage is technically unauthorized reselling — if user base exceeds ~50, require users to connect their own Apollo API key

7. **Token → "Credits" wording**: In settings, replace "token balance" with "credits" or similar user-friendly wording.

### Process & Collaboration Decisions

8. **PRD to Git**: Move PRD and User Stories from Google Docs into the git repo as `.md` files. Highlight outdated sections for manual review.

9. **Remove HANDOFF**: Traces system covers handoff notes, so HANDOFF process is paused.

10. **UX testing for Heyzeva**: Celine to do UX testing of Heyzeva's staging build from an agency persona perspective (dental marketing agency managing multiple client sites). Provide feedback via git (md files).

11. **Work split this week**: ~60% Gallop, ~40% Heyzeva UX review.

12. **Use "Whi" app**: Voice-to-text app for capturing thoughts, then have Claude Code convert transcripts to structured md files.
